Access Profiles

Use these roles to decide what each staff member should be able to do inside the dashboard.

Owner / Admin Full dashboard access
Can access dashboard, events, revenue, members, messages, contact log, settings, and check-in.
Manager Operations access
Can access daily operations, event management, guest lists, members, messages, and check-in.
Door Staff Check-in only
Can only access guest lists, QR scanning, and check-in tools.
Bookkeeper Revenue-focused access
Can access revenue, expenses, memberships, ticket sales, and reporting tools.
